# CC-15: Frontend UI

## Overview
Complete React frontend for the platform.

## Priority: P0

## Pages
- Dashboard (metrics, recent activity)
- Contacts (list, detail, edit)
- Conversations (inbox view)
- Workflows (list, builder)
- Campaigns (list, create)
- Settings (account, integrations)

## Tech Stack
- React 18 + TypeScript
- TailwindCSS
- React Query
- React Router
- Zustand (state)

## Success Criteria
- [ ] All pages functional
- [ ] Responsive design
- [ ] Real-time updates (WebSocket)
- [ ] Command palette (Cmd+K)
- [ ] Dark mode (stretch)
